Tier 1 Program Brief

Transformational Programs | Investment: Typically $5M+ | Cycle Time: 6-12 weeks

Important: Tier 1 Uses a Different Process

Tier 1 programs are transformational investments that require comprehensive planning with Strategic Planning Office support.

This is NOT a quick web form. You will need to download a template and work with the Strategy Office to complete a 10-page Program Brief.

Tier 1 Characteristics

When is Tier 1 Required?

  • Investment >$2M with medium/high complexity
  • OR Investment $500k-$2M with high complexity
  • Transformational change with enterprise-wide impact
  • Multi-year strategic initiatives
  • Programs requiring board-level visibility (especially >$10M)

Value Requirements

Strategic Imperative: Must directly enable 3+ strategic objectives; board-level priority
Transformational Value: Significant value across ALL FIVE categories
Financial Return: NPV >$5M over 7 years; IRR >15%
Enterprise Impact: Cross-domain transformation; organizational change

Required Documentation

Program Brief (10 pages)
  • Executive Summary
  • Strategic Context
  • Business Need
  • Proposed Approach
  • Expected Value
  • Costs and Funding
  • Risks and Dependencies
  • Decision Request
Supporting Appendices
  • Financial Model (detailed)
  • Regulatory Documentation
  • Stakeholder Analysis
  • High-Level Architecture
  • Vendor Proposals/Quotes
  • Benchmarking Research

Governance Process

Six-Gate Process

  1. Gate 0: Program Brief Review (Initial screening)
  2. Gate 1: Qualification & Business Case Development (4-8 weeks)
  3. Gate 2: Architecture Board Review (7-10 days)
  4. Gate 3: Technology Subcommittee Deep Dive (2-3 hour session)
  5. Gate 4: ELT Pre-Briefing (Socialize with executives)
  6. Gate 5: TAC Review and Funding Authorization
  7. Gate 6: Board Review (if >$10M or strategic significance)

Total Cycle Time: 6-12 weeks
